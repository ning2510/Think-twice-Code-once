#### 面试题 01.01. 判定字符是否唯一

**位运算/状态压缩**

首先我们可以看出来给定的字符串都是小写字母，共26个即26个状态，int类型足以存放这26个状态

- 通过 & 操作，我们可以去判断某个状态我们是否做过标记

> 例：`x & (1 << 0)` 判断第一个状态是否已经做过标记，若做过标记，说明状态重复，返回false

- 通过 | 操作，我们可以去给某个状态做标记

> 例：`x | (1 << 0)` 给第一个状态做标记

具体操作见代码

 

**代码 :**

 ```c
class Solution {
public:
    bool isUnique(string astr) {
        int x = 0;
        for(int i = 0; i < astr.size(); i++) {
            if(x & (1 << (astr[i] - 'a'))) return false;
            else x |= (1 << (astr[i] - 'a'));
        }
        return true;
    }
};
 ```
